# Getting Started

## What is this website?
This website is a **Bus Tracking and Management System**.  
It helps students, drivers, and admins track buses, manage routes, and receive important notices.

There are **6 bus routes** in total:
- **3 Regular routes** (daily buses)
- **3 Occasional routes** (special or limited-time buses)
