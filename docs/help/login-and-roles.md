# Login and User Roles

## How to login
1. Go to the **Login** page
2. Enter your email and password
3. Click **Login**
4. You will be redirected to your dashboard

---

## User Roles

### Student
Students have read-only access.
They can:
- View all bus routes
- Track buses on the map
- Check bus status
- Read notices

Students **cannot**:
- Add or edit buses
- Modify routes
- Publish notices

---

### Driver
Drivers are responsible for live tracking.
They can:
- View their assigned bus
- See assigned route and stops
- Share live location

Drivers **cannot**:
- Add new routes
- Assign buses
- Manage users

---

### Admin
Admins have full access.
They can:
- Add, edit, and delete buses
- Create and manage routes
- Assign routes to buses
- Publish and delete notices
- Manage drivers and users
