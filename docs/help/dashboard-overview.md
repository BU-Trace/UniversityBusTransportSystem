# Dashboard Overview

## Dashboard Layout
After login, you will see a dashboard with a sidebar menu.

Common sections:
- Bus Management
- Route Management
- Driver Management
- Notices
- Logout

The available sections depend on your role.

---

## Student Dashboard
Students see:
- Route list
- Bus tracking map
- Notice button
- Current bus status

Students mainly use the dashboard to **check buses and routes**.

---

## Driver Dashboard
Drivers see:
- Assigned bus details
- Route information
- Live location status

Drivers mainly use the dashboard to **share bus location**.

---

## Admin Dashboard
Admins see all sections:
- Bus Manage
- Route Manage
- Driver Manage
- Notice Board

Admins use the dashboard to **control the whole system**.
