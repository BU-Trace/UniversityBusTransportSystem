# Route Management
## What is a route?
- List of stops
1. Go to **Route Manage**
4. Select route type
6. Enter stop names
7. Add Google Map location if required
8. Save the route

Students can:
- View all available routes
- See route stops
- Check which buses run on each route

---

## How to check routes as a student
1. Open the dashboard
2. Go to routes section
3. Click a route
4. View stops and bus details
