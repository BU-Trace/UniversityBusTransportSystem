# Route Management
## What is a route?
A route defines:
- Type (Regular or Occasional)
- List of stops
- Map path

There are **6 routes**:
- 3 Regular
- 3 Occasional

---

## How admin creates a route
1. Go to **Route Manage**
2. Click **Add Route**
3. Enter route name
4. Select route type
5. Select number of stops
6. Enter stop names
7. Add Google Map location if required
8. Save the route

---

## How students use routes
Students can:
- View all available routes
- See route stops
- Check which buses run on each route

---

## How to check routes as a student
1. Open the dashboard
2. Go to routes section
3. Click a route
4. View stops and bus details
